## Getting Started

### Prerequisites
- Node.js
- npm

#### Backend
The backend runs on port -3333
1.  Navigate into the Backend
		```
		cd backend
		```
2.  Install dependencies
		```
		npm install
		```
3. Run Backend Tests
		 ```
		npm run test
		```
4. Run Backend server
		 ```
		npm run dev
		```

#### Frontend 
The backend runs on port -4444
1.  Navigate into the Frontend  
		```
		cd frontend
		```
2.  Install dependencies
		```
		npm install
		```
3. Run Frontend Tests
		 ```
		npm run test
		```
4. Run Frontend app
		 ```
		npm run dev
		```
